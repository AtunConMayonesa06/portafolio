# Portafolio - Despliegue en GitHub Pages


Este repositorio contiene un portafolio estático (HTML/CSS) listo para publicarse en **GitHub Pages**.


## Pasos rápidos para publicar
1. Asegúrate de que la rama principal se llame `main`.
2. Coloca los ZIPs con EXACTAMENTE estos nombres dentro de la carpeta `descargas/`:
- `ejercicio1.zip`
- `Investigacion 1.zip`
- `Investigacion 3.zip`
- `laboratorio 1.zip`
- `laboratorio 2.zip`
- `Parcial 2.zip`
- `parcial1.zip`
- `taller 1.zip`
- `taller3.zip`
3. Haz commit y push a GitHub.


```bash
git add .
git commit -m "Agregar portafolio estático y páginas de proyectos"
git push origin main
```


4. En GitHub ve al repositorio → Settings → Pages → Source → selecciona `main` branch y carpeta `/ (root)` → Save.
5. Espera 1–2 minutos y abre: `https://AtunConMayonesa06.github.io/portafolio/` (o `https://AtunConMayonesa06.github.io/` si es repo usuario-site).


## Notas
- Los ZIP deben subirse al repositorio (carpeta `descargas`) para que el botón de descarga funcione directamente.
- Si prefieres alojar los ZIP en otro lugar (Drive, Dropbox), cambia los hrefs en `proyectos.html` y las páginas individuales.
